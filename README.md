CityAdaptAI is an AI‑driven smart city platform that provides
context‑aware recommendations, real‑time services, and intelligent
user interaction through a mobile application.

## Features
- AI‑Powered Chat Assistant
- Context‑Aware Recommendations
- Real‑Time Event and Service Updates
- Smart City Service Discovery
- User‑Friendly Mobile Interface

## Technologies Used
- Frontend: React Native (Expo)
- Backend: FastAPI (Python)
- Machine Learning: TensorFlow / PyTorch
- Data Handling: Pandas, NumPy

## Project Type
Capstone / Final Year Project
